# 0x06. Unittests in JS

### Project setup
```
cd 0x06-unittests_in_js
npm install mocha --save-dev
# Set up a test script in package.json:
cat  package.json
 "scripts": {
   "test": "mocha"
 }
# run tests
npm run test *.test.js
```
